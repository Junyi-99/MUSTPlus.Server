## **获取当前周数**

  获取当前周是该学期第几周

- **URL**

  _/basic/week_

- **Method**

  `GET`
  
- **REST Params**
  
  None
  
- **URL Params**

  **Required**

  None （无加密要求）

  **Optional**

  None

- **Data Params**

  None

- **Success Response:**

  ```JSON
  {"code":0,"msg":"","week":2}
  ```

- **Error Response:**

  没有错误信息
